# First Order 
## Reconstruction Method
First order upwind method
$U_i^{n+1}=U_i^n-\frac{\Delta t}{\Delta x}(F^n_{i+1/2}-F^n_{i-1/2})$
where
$F^i_{n+1/2}=\frac{U^i_{n+1}-U^i_n}{\Delta x}$
